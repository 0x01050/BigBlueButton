Desktop Sharing Help

This page contains a list of videos that let you quickly get started using BigBlueButton's desktop sharing.

Desktop sharing requires Java 1.8. If you don't have Java 1.8 installed (or are not sure if you have it installed), first visit java.com and install Java.
